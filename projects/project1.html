Badge Generator

GitHub Live Demo

Overview

Система генерации и управления бейджами с QR-кодами для продавцов. Проект включает в себя веб-интерфейс для создания бейджей, систему аутентификации, автоматическую генерацию QR-кодов и интеграцию с базой данных продавцов. Система обеспечивает безопасное хранение данных и удобный процесс верификации.

Features

  • Генерация уникальных QR-кодов для бейджей
  • Интеграция с базой данных продавцов
  • Безопасная аутентификация и авторизация
  • Автоматическое создание и сохранение бейджей
  • HTTPS-защита и SSL-сертификация
  • Контейнеризация с Docker для простого развертывания

Technologies Used

  • Python (Flask) для бэкенда
  • PostgreSQL для хранения данных
  • Docker для контейнеризации
  • Nginx для проксирования и SSL
  • qrcode для генерации QR-кодов
